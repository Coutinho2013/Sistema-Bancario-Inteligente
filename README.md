# **Sistema Bancário Inteligente**
Esse programa permite você criar uma conta, poder sacar, depositar,
e ver o extrato bancário mostrando seu nome, gênero e idade
(só podendo criar a conta maiores de 12), quanto você tem de saldo,
quantas vezes você depositou e o valor total dos depositos que você já fez,
quantas vezes você sacou e o valor total de todos os saques (limite de até 3 saques por programa).
- Pode criar uma conta
- Pode sacar e depositar dinheiro
- Pode ver o extrato bancário da conta
   - **Extrato**
      - Nome
      - Idade (Para maiores de 12 anos)
      - Saldo da conta
      - Total e quantidades de todos os depósitos
      - Total e quantidades (Limite de até 3) de todos os saques

## Comentário
 É um programa feito da linguagem python
 por um estudante de 12 anos para fins
 de aprendizado sobre lógica e programação.
   
## Objetivo
Esse programa foi feito para estudos de lógica e desenvolvimento de python.

## Linguagem
Python 3.13+.
